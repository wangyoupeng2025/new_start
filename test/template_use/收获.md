# 模板学习笔记

## 一、模板的核心概念

### 1. 什么是模板？

`template` 关键字声明要创建一个模板。模板是 C++ 中实现**泛型编程**的工具，让你写出**与类型无关**的代码。

### 2. 模板的核心价值

| 价值 | 说明 |
|------|------|
| 提高代码复用性 | 一份代码，适用于多种类型 |
| 关注逻辑，不关注类型 | 只注重要实现的内容，无需关心传入的数据类型 |
| 类型安全 | 编译期检查类型 |

### 3. 基本语法

二、函数模板

基本用法

cpp
#include <iostream>
using namespace std;

// 函数模板：比较两个值，返回较大的
template<typename T>
T myMax(T a, T b) {
    return a > b ? a : b;
}

int main() {
    cout << myMax(3, 5) << endl;        // 5（T = int）
    cout << myMax(3.14, 2.71) << endl;  // 3.14（T = double）
    cout << myMax('c', 'a') << endl;    // c（T = char）
    cout << myMax(string("hello"), string("world")) << endl; // world
    return 0;
}

三、类模板

基本用法

cpp
// 类模板：可以存储任意类型的盒子
template<typename T>
class Box {
private:
    T value;
public:
    void set(T v) { value = v; }
    T get() { return value; }
};

int main() {
    Box<int> intBox;
    intBox.set(100);
    cout << intBox.get() << endl;  // 100

    Box<string> strBox;
    strBox.set("hello");
    cout << strBox.get() << endl;  // hello

    Box<double> doubleBox;
    doubleBox.set(3.14);
    cout << doubleBox.get() << endl;  // 3.14

    return 0;
}
类模板成员函数类外定义

cpp
template<typename T>
class Box {
private:
    T value;
public:
    void set(T v);
    T get();
};

// 类外定义也要加 template
template<typename T>
void Box<T>::set(T v) {
    value = v;
}

template<typename T>
T Box<T>::get() {
    return value;
}
多个模板参数

cpp
template<typename K, typename V>
class Pair {
private:
    K key;
    V value;

    Pair(K k, V v) : key(k), value(v) {}

四、模板特化

1. 什么是模板特化？

当某些特殊数据类型用默认模板会出问题时，需要模板特化。直接将对这种特殊类型的数据处理的逻辑单独写出来。

2. 为什么需要特化？

通用模板对大多数类型工作良好，但对某些特殊类型可能：

逻辑不正确（如浮点数比较）
效率低下（如 bool 数组）
语法不支持（如某些操作符未重载）
3. 函数模板特化

cpp
#include <iostream>
#include <string>
#include <cmath>
using namespace std;

// 通用模板：默认处理
template<typename T>
void print(T value) {
    cout << "普通: " << value << endl;
}

// 特化：int 类型特殊处理
template<>
void print<int>(int value) {
    cout << "整数: " << value << endl;
}

// 特化：bool 类型特殊处理（规避 0/1 输出问题）
template<>
void print<bool>(bool value) {
    cout << "布尔: " << (value ? "真" : "假") << endl;
}

// 特化：string 类型特殊处理
template<>
void print<string>(string value) {
    cout << "字符串: " << value << "，长度=" << value.length() << endl;
}

// 比较函数的特化：浮点数不能用 == 直接比较
template<typename T>
bool compare(T a, T b) {
    return a == b;
}

template<>
bool compare<double>(double a, double b) {
    return abs(a - b) < 0.000001;  // 误差范围内算相等
}

int main() {
    print(10);           // 整数: 10（走特化）
    print(3.14);         // 普通: 3.14（走通用）
    print(true);         // 布尔: 真（走特化，规避 0/1）
    print("hello");      // 普通: hello（走通用）
    print(string("world"));  // 字符串: world，长度=5（走特化）
    
    cout << compare(1.0, 1.0) << endl;      // 1（true）
    cout << compare(0.1 + 0.2, 0.3) << endl; // 1（true，浮点误差处理）
    
    return 0;
}
4. 类模板特化

cpp
#include <iostream>
#include <vector>
using namespace std;

// 通用模板：普通存储
template<typename T>
class Storage {
private:
    T data;
public:
    Storage(T value) : data(value) {}
    void print() { cout << "普通存储: " << data << endl; }
};

// 特化：bool 类型用位存储（节省内存）
template<>
class Storage<bool> {
private:
    unsigned char bits;  // 用位存储
public:
    Storage(bool value) {
        bits = value ? 1 : 0;
    }
    void print() {
        cout << "布尔存储: " << (bits ? "true" : "false") << endl;
    }
};

// 特化：vector 类型特殊处理
template<typename T>
class Storage<vector<T>> {
private:
    vector<T> data;
public:
    Storage(vector<T> v) : data(v) {}
    void print() {
        cout << "vector存储，大小=" << data.size() << endl;
    }
};

int main() {
    Storage<int> s1(100);
    s1.print();  // 普通存储: 100
    
    Storage<bool> s2(true);
    s2.print();  // 布尔存储: true（特化版本）
    
    Storage<bool> s3(false);
    s3.print();  // 布尔存储: false
    
    vector<int> v = {1, 2, 3};
    Storage<vector<int>> s4(v);
    s4.print();  // vector存储，大小=3
    
    return 0;
}
5. 特化 vs 普通模板对比

对比	普通模板	特化版本
适用类型	大多数类型	特定特殊类型
实现	通用逻辑	专门优化/修正
触发条件	类型不匹配特化时	类型完全匹配特化时
五、模板与 STL

STL 容器和算法都是模板的典型应用：

cpp
// vector 是类模板
vector<int> vi;        // 存储 int
vector<string> vs;     // 存储 string
vector<Student> vstu;  // 存储自定义类型

// sort 是函数模板
sort(vi.begin(), vi.end());  // 适用于任何支持随机访问的容器

// map 有两个模板参数
map<string, int> scores;  // key 是 string，value 是 int
