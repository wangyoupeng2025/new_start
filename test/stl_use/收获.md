# STL 容器学习笔记

## 一、vector（动态数组）

### 1. 基本概念

```cpp
vector<数据类型> 容器名;
长度可变，动态增长
内存连续，支持随机访问
尾部插入/删除效率高，中间插入/删除效率低
2. 遍历方法

cpp
// 迭代器遍历
for (auto it = v.begin(); it != v.end(); ++it) {
    cout << *it << endl;
}

// 范围 for 循环（推荐）
for (const auto& x : v) {
    cout << x << endl;
}

// 下标遍历（需要随机访问）
for (size_t i = 0; i < v.size(); i++) {
    cout << v[i] << endl;
}
3. 核心函数

函数	说明	时间复杂度
push_back(val)	尾部插入值	O(1) 均摊
pop_back()	删除尾部元素	O(1)
emplace_back(args...)	尾部直接构造	O(1) 均摊
insert(pos, val)	指定位置插入	O(n)
erase(pos)	删除指定位置	O(n)
clear()	清空所有元素	O(n)
size()	返回元素个数	O(1)
empty()	判断是否为空	O(1)
reserve(n)	预分配容量	O(n)
resize(n)	改变大小	O(n)
at(pos)	边界检查访问	O(1)
operator[](pos)	下标访问（无检查）	O(1)
front()	访问第一个元素	O(1)
back()	访问最后一个元素	O(1)
begin() / end()	获取迭代器	O(1)
4. 下标访问

cpp
vector<int> v = {10, 20, 30, 40, 50};

// 方式1：operator[]（无边界检查，更快）
cout << v[0] << endl;   // 10
cout << v[2] << endl;   // 30
v[1] = 25;              // 修改

// 方式2：at()（有边界检查，更安全）

// cout << v.at(10) << endl;  // 抛出 out_of_range 异常

// 方式3：下标遍历
for (size_t i = 0; i < v.size(); i++) {
    cout << v[i] << " ";
}
5. 增删改查示例

cpp
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> v;
    
    // 增
    v.push_back(10);           // 尾部添加
    v.push_back(20);
    v.push_back(30);
    v.insert(v.begin(), 5);    // 头部插入：5,10,20,30
    v.insert(v.begin() + 2, 15); // 位置2插入：5,10,15,20,30
    
    // 查
    int val = v[2];            // 下标访问：15
    int front = v.front();     // 第一个元素：5
    int back = v.back();       // 最后一个元素：30
    
    // 查找元素位置（需要 algorithm）
    auto it = find(v.begin(), v.end(), 20);
    if (it != v.end()) {
        cout << "找到了: " << *it << endl;
    }
    
    // 改
    v[1] = 100;                // 修改下标1的元素
    
    // 删
    v.pop_back();              // 删除尾部
    v.erase(v.begin());        // 删除头部
    v.erase(v.begin() + 1);    // 删除位置1
    v.clear();                 // 清空所有
    
    return 0;
}
6. emplace_back 详解

cpp
vector<Student> v;

// push_back：先构造临时对象，再移动/复制到 vector
v.push_back(Student("小明", 18));
// 步骤：临时对象构造 → 复制/移动 → 临时对象析构

// emplace_back：直接在 vector 内部构造对象
v.emplace_back("小明", 18);
// 步骤：直接在 vector 内部构造，一步完成
特性	push_back	emplace_back
参数	接受对象	接受构造参数
临时对象	可能创建	不创建
构造次数	1次 + 可能移动/复制	1次
效率	较低	较高
使用场景	对象已存在	直接构造新对象
7. vector<bool> 特化

cpp
vector<bool> flags(10, true);  // 10个布尔值，全为true
flags[0] = false;              // 将下标0改为false
特点：

每个元素仅占 1 位（bit），而非 1 字节
节省内存（8倍压缩）
不能取元素地址，不能绑定 bool& 引用
场景	推荐
需要节省内存（大量 bool）	✅ 使用 vector<bool>
需要取地址	❌ 使用 vector<char>
需要引用绑定	❌ 使用 vector<char>
需要位操作	✅ 使用 vector<bool>
二、map 和 unordered_map

1. 基本概念

cpp
#include <map>
#include <unordered_map>

// map：有序（红黑树），O(log n)
map<string, int> scores;

// unordered_map：无序（哈希表），O(1) 平均
unordered_map<string, int> scores;
存储键值对（key-value）
有序是指根据 key 的大小自动排序，而非 value
每个 key 唯一
2. 遍历方法

cpp
// 结构化绑定遍历（C++17）
for (const auto& [name, score] : scores) {
    cout << name << ": " << score << endl;
}

// 迭代器遍历
for (auto it = scores.begin(); it != scores.end(); ++it) {
    cout << it->first << ": " << it->second << endl;
}

// 范围 for + pair
for (const auto& pair : scores) {
    cout << pair.first << ": " << pair.second << endl;
}
3. 语法拆解

cpp
for (const auto& [k, v] : um)
//   ↑       ↑    ↑    ↑
//   │       │    │    └── 要遍历的容器
//   │       │    └── 绑定变量（k接收key，v接收value）
//   │       └── 类型推导（auto）
//   └── const 修饰（不可修改），& 引用（避免拷贝）
4. 核心函数

函数	说明	时间复杂度
insert({key, value})	插入键值对	O(log n) / O(1)
emplace(key, value)	直接构造插入	O(log n) / O(1)
try_emplace(key, args...)	不存在时构造	O(log n) / O(1)
erase(key)	删除指定 key	O(log n) / O(1)
erase(it)	删除迭代器位置	O(1) 均摊
find(key)	查找，返回迭代器	O(log n) / O(1)
count(key)	返回 key 数量（0或1）	O(log n) / O(1)
contains(key) (C++20)	判断是否存在	O(log n) / O(1)
clear()	清空	O(n)
size()	返回元素个数	O(1)
empty()	判断是否为空	O(1)
5. 增删改查示例

cpp
map<string, int> scores;

// 增
scores["小明"] = 100;                    // 方式1：[] 操作符
scores.insert({"小红", 95});             // 方式2：insert
scores.emplace("小刚", 87);              // 方式3：emplace
scores.try_emplace("小李", 92);          // 方式4：try_emplace（C++17）

// 查
// 方式1：find（推荐）
auto it = scores.find("小明");
if (it != scores.end()) {
    cout << it->first << ": " << it->second << endl;
}

// 方式2：[]（不推荐用于查找，会插入默认值）
int val = scores["小明"];  // 存在则返回，不存在则插入0

// 方式3：at（有边界检查）
try {
    int val = scores.at("小明");
} catch (const out_of_range& e) {
    cout << "key 不存在" << endl;
}

// 方式4：contains（C++20）
if (scores.contains("小明")) {
    cout << "存在" << endl;
}

// 改
scores["小明"] = 98;                     // 直接赋值修改
auto it2 = scores.find("小红");
if (it2 != scores.end()) {
    it2->second = 96;                    // 通过迭代器修改
}

// 删
scores.erase("小刚");                    // 按 key 删除
scores.erase(scores.begin());            // 按迭代器删除
scores.clear();                          // 清空
6. find() vs [] 区别

特性	map.find(key)	map[key]
返回值	迭代器	value 的引用
key 不存在时	返回 end()	自动插入默认值
是否会修改 map	❌ 不会	✅ 会（插入新元素）
查找效率	O(log n)	O(log n)
const map 可用	✅ 可用	❌ 不可用（会修改）
推荐用于	纯粹的查找	插入/修改
cpp
map<string, int> m;

// find：不会修改 map
auto it = m.find("不存在");  // it == m.end()
cout << m.size();           // 0

// []：会插入默认值
int val = m["不存在"];       // 插入 {"不存在", 0}
cout << m.size();           // 1
7. 下标访问限制

cpp
// ❌ map 不支持整数下标（除非 key 是 int 类型）
// cout << m[0] << endl;  // 如果 key 是 string，这行会编译错误

// ✅ 只有 key 类型匹配才能用 []

三、set 和 unordered_set

1. 基本概念

cpp
#include <set>

// set：有序，不重复，红黑树
set<int> numbers;

// unordered_set：无序，不重复，哈希表

存储唯一值（自动去重）
一个值最多出现一次
set 会根据值的大小自动排序
2. 遍历方法

cpp
set<int> numbers = {5, 2, 8, 1, 9};

// 范围 for
for (int x : numbers) {
    cout << x << " ";  // 输出：1 2 5 8 9（自动排序）

// 迭代器
for (auto it = numbers.begin(); it != numbers.end(); ++it) {
    cout << *it << " ";
}
3. 核心函数

函数	说明	时间复杂度
insert(val)	插入值（自动去重）	O(log n) / O(1)
emplace(args...)	直接构造插入	O(log n) / O(1)
erase(val)	删除指定值	O(log n) / O(1)

find(val)	查找，返回迭代器	O(log n) / O(1)
count(val)	返回数量（0或1）	O(log n) / O(1)
contains(val) (C++20)	判断是否存在	O(log n) / O(1)

size()	返回元素个数	O(1)

cpp
set<int> s;

// 增

cpp
list<int> lst;

// 增
lst.push_back(10);     // 尾部：10
lst.push_front(5);     // 头部：5,10
lst.push_back(20);     // 尾部：5,10,20
lst.insert(++lst.begin(), 7);  // 位置1插入：5,7,10,20

// 删
lst.pop_back();    // 删除尾部

5. deque 下标访问原理

text
deque 底层是分段连续内存：

[缓冲区0] [缓冲区1] [缓冲区2] [缓冲区3]
   ↓         ↓         ↓         ↓
  [0-7]     [8-15]    [16-23]   [24-31]

operator[] 通过中央控制器计算：
index → 缓冲区索引 + 缓冲区内偏移
时间复杂度 O(1)
六、容器选择指南

场景	推荐容器	原因
需要频繁头部插入/删除	deque	O(1) 头部操作
需要频繁尾部插入/删除	vector 或 deque	都是 O(1)
需要频繁中间插入/删除	list	O(1) 中间操作
需要随机访问	vector 或 deque	O(1) 访问
需要内存连续性	vector	连续内存，缓存友好
需要在两端操作	deque	push_front/pop_front
需要键值对查找	map / unordered_map	O(log n) / O(1)
需要去重存储	set / unordered_set	自动去重
大量 bool 值	vector<bool>	内存压缩
七、总结对比表

容器	随机访问	头部插入	尾部插入	中间插入	内存连续性
vector	✅ O(1)	❌ O(n)	✅ O(1)	❌ O(n)	连续
deque	✅ O(1)	✅ O(1)	✅ O(1)	❌ O(n)	分段连续
list	❌	✅ O(1)	✅ O(1)	✅ O(1)	不连续
map	❌	-	-	-	树结构
