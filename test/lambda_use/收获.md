# Lambda 表达式学习笔记

## 一、基本结构

```cpp
[](){}   // 捕获列表 + 参数列表 + 函数体
二、捕获列表 []

1. 不捕获

cpp
[]      // 不从外部获取任何变量
2. 单个变量捕获

写法	含义	能否修改外部
[x]	值捕获变量 x（副本，值传递）	不能
[&x]	引用捕获变量 x（用本身）	能
cpp
int x = 10;

// 值捕获：不能修改外部
auto f1 = [x]() { 
    // x = 20;  // 编译错误
    cout << x << endl;  // 可以读取
};

// 引用捕获：可以修改外部
auto f2 = [&x]() { 
    x = 20;  // 修改外部变量
};
3. 全部捕获

写法	含义	能否修改外部
[=]	所有用到的变量都是值传递	不能
[&]	所有用到的变量都是引用传递	能
编译器自动分析函数体，只捕获用到的变量。

cpp
int a = 1, b = 2, c = 3;

// 只用了 a 和 b，编译器自动捕获 a 和 b
auto f1 = [=]() {
    cout << a + b << endl;  // c 没用到，不捕获
};

// 引用捕获所有用到的变量
auto f2 = [&]() {
    a = 10;
    b = 20;  // 可以修改
};
4. 混合捕获

写法	含义
[=, &x]	默认值捕获，x 引用捕获
[&, x]	默认引用捕获，x 值捕获
cpp
int a = 1, b = 2, x = 10;

// 默认值捕获，x 引用捕获
auto f1 = [=, &x]() {
    // a, b 是值捕获（不能改）
    // x 是引用捕获（可以改）
    x = 100;   // 可以
    // a = 200;  // 错误
};

// 默认引用捕获，x 值捕获
auto f2 = [&, x]() {
    // a, b 是引用捕获（可以改）
    // x 是值捕获（不能改）
    a = 200;   // 可以
    // x = 100;  // 错误
};
三、参数列表 ()

和普通函数一样，可以传参数。

写法	含义
(int x)	值传递
(int& x)	引用传递
cpp
// 值传递：修改不影响原数据
for_each(v.begin(), v.end(), [](int x) {
    x = 100;  // 只修改副本，原数组不变
});

// 引用传递：修改原数据
for_each(v.begin(), v.end(), [](int& x) {
    x = 100;  // 修改原数组元素
});
参数数量由调用的算法决定：

for_each 每次传 1 个元素 → Lambda 接收 1 个参数
sort 的比较函数需要 2 个参数 → Lambda 接收 2 个参数
四、函数体 {}

写具体的实现逻辑。

cpp
auto add = [](int a, int b) {
    return a + b;  // 函数体
};
五、优缺点

优点：

就近编写逻辑，代码集中
无需像普通函数那样麻烦
可捕获外部变量
缺点：

匿名函数，无法复用
复杂逻辑时可读性下降
六、复用方式

给 Lambda 起个名字，就可以复用了。

cpp
// 有名字，可以复用
auto add = [](int a, int b) {
    return a + b;
};

cout << add(1, 2) << endl;  // 3
cout << add(3, 4) << endl;  // 7
七、配合算法使用（常见场景）

cpp
vector<int> v = {1, 2, 3, 4, 5};

// 累加求和（引用捕获 sum）
int sum = 0;
for_each(v.begin(), v.end(), [&sum](int x) {
    sum += x;
});
cout << sum << endl;  // 15

// 条件查找
auto it = find_if(v.begin(), v.end(), [](int x) {
    return x > 3;
});
cout << *it << endl;  // 4

// 自定义排序
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // 降序
});
八、捕获 vs 参数 对比

对比项	捕获变量 [&sum]	参数变量 (int x)
数据来源	Lambda 外部的变量	调用时传入的值
传递方式	值/引用（由捕获方式决定）	值传递（除非用 int&）
能否修改外部	引用捕获可以	不能
每次调用	同一个变量	重新传入
cpp
int sum = 0;
for_each(v.begin(), v.end(), [&sum](int x) {
    sum += x;   // sum 引用捕获，可修改外部
    // x 值传递，修改不影响原数组
});
九、快速参考表

写法	含义
[]	不捕获任何变量
[x]	值捕获 x
[&x]	引用捕获 x
[=]	所有用到的变量值捕获
[&]	所有用到的变量引用捕获
[=, &x]	默认值捕获，x 引用捕获
[&, x]	默认引用捕获，x 值捕获
(int x)	参数值传递
(int& x)	参数引用传递
十、一句话总结

Lambda 是匿名函数，结构为 {}
[] 定义捕获方式（值/引用）
() 定义参数
{} 定义逻辑
优点：就近编写，代码简洁
配合 STL 算法使用效果最佳
