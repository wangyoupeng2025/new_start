# STL 算法学习笔记

## 一、常用算法速查表

| 算法类型 | 作用 | Lambda 写法模式 |
|---------|------|----------------|
| 查找 `find_if` | 按条件查找第一个元素 | `[](int x) { return 条件; }` |
| 计数 `count_if` | 按条件统计个数 | `[](int x) { return 条件; }` |
| 排序 `sort` | 排序 | `[](int a, int b) { return a > b; }` |
| 遍历 `for_each` | 遍历每个元素 | `[](int x) { 操作; }` 或 `[&](int x) { 操作; }` |
| 转换 `transform` | 转换每个元素 | `[](int x) { return 新值; }` |
| 删除 `remove_if` | 条件删除 | `[](int x) { return 删除条件; }` |

**注意：**
- 这些函数名是标准库写好的，不能自己改
- 一般配合 Lambda 使用
- 有些需要变量接收返回值，否则没意义

## 二、查找 `find_if`

### 基本语法

```cpp
auto it = find_if(v.begin(), v.end(), [](int x) {
    return x >= 2;  // 条件
});

if (it != v.end()) {
    cout << "找到了: " << *it << endl;
}
返回值

找到：返回指向该元素的迭代器
没找到：返回 v.end()
必须检查返回值

cpp
// ❌ 错误：没有使用返回值，会产生 nodiscard 警告
find_if(v.begin(), v.end(), [](int x) { return x > 5; });

// ✅ 正确：接收返回值并检查
auto it = find_if(v.begin(), v.end(), [](int x) { return x > 5; });
if (it != v.end()) {
    cout << *it << endl;
}
三、计数 count_if

基本语法

cpp
int cnt = count_if(v.begin(), v.end(), [](int x) {
    return x % 2 == 0;  // 统计偶数个数
});
cout << "偶数个数: " << cnt << endl;
返回值

返回满足条件的元素个数（int 类型）

四、排序 sort

升序排序（默认）

cpp
sort(v.begin(), v.end());
降序排序（Lambda）

cpp
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // a > b 时 a 排在前面
});
自定义排序

cpp
// 按绝对值排序
sort(v.begin(), v.end(), [](int a, int b) {
    return abs(a) < abs(b);
});

// 按字符串长度排序
sort(strs.begin(), strs.end(), [](const string& a, const string& b) {
    return a.length() < b.length();
});
五、遍历 for_each

只读遍历

cpp
for_each(v.begin(), v.end(), [](int x) {
    cout << x << " ";
});
修改原值（引用捕获）

cpp
int sum = 0;
for_each(v.begin(), v.end(), [&sum](int x) {
    sum += x;  // 累加到外部变量
});
修改容器元素（引用参数）

cpp
for_each(v.begin(), v.end(), [](int& x) {
    x = x * 2;  // 修改原数组
});
六、转换 transform

方式1：结果写入新容器（源数据不变）

cpp
vector<int> v = {1, 2, 3, 4, 5};
vector<int> result(v.size());

transform(v.begin(), v.end(), result.begin(), [](int x) {
    return x * 2;
});
// v 仍是 {1, 2, 3, 4, 5}
// result 变成 {2, 4, 6, 8, 10}
方式2：结果写回原容器（原地修改）

cpp
vector<int> v = {1, 2, 3, 4, 5};

transform(v.begin(), v.end(), v.begin(), [](int x) {
    return x * 2;
});
// v 变成 {2, 4, 6, 8, 10}
方式3：两个输入源

cpp
vector<int> v1 = {1, 2, 3};
vector<int> v2 = {4, 5, 6};
vector<int> result(v1.size());

transform(v1.begin(), v1.end(), v2.begin(), result.begin(), [](int a, int b) {
    return a + b;  // 对应位置相加
});
// result: {5, 7, 9}
transform vs for_each

对比	transform	for_each
目的	计算新值	执行操作
返回值	有返回值	无返回值
修改原值	通过返回值写入	通过引用参数
七、删除 remove_if + erase（erase-remove 惯用法）

完整写法

cpp
v.erase(remove_if(v.begin(), v.end(), [](int x) {
    return x % 2 == 0;  // 删除所有偶数
}), v.end());
分步拆解

第1步：remove_if - 把要删除的元素移到末尾

cpp
auto newEnd = remove_if(v.begin(), v.end(), [](int x) {
    return x % 2 == 0;
});
remove_if 做了什么：

text
原始 v: [1, 2, 3, 4, 5, 6]
           ↑  ↑     ↑  ↑
         偶数 偶数  偶数 偶数

remove_if 后:
v: [1, 3, 5, 2, 4, 6]
           ↑
        newEnd 指向这里（第一个"垃圾"位置）
把要保留的元素移到前面
把要删除的元素移到后面（垃圾区）
返回指向第一个垃圾元素的迭代器
第2步：erase - 真正删除垃圾区

cpp
v.erase(newEnd, v.end());
删除从 newEnd 到末尾的所有元素（即垃圾区）。

图解全过程

text
初始:
[1, 2, 3, 4, 5, 6]

remove_if 后（元素被移动，大小不变）:
[1, 3, 5, 2, 4, 6]
           ↑
        newEnd（指向第一个垃圾）

erase 后（真正删除）:
[1, 3, 5]
其他删除场景

需求	写法
删除所有等于 5 的元素	v.erase(remove(v.begin(), v.end(), 5), v.end())
删除所有偶数	v.erase(remove_if(v.begin(), v.end(), [](int x){ return x%2==0; }), v.end())
删除所有大于 10 的元素	v.erase(remove_if(v.begin(), v.end(), [](int x){ return x>10; }), v.end())
为什么不能直接 erase？

cpp
// ❌ 错误：不能在遍历时直接 erase
for (auto it = v.begin(); it != v.end(); ++it) {
    if (*it % 2 == 0) {
        v.erase(it);  // 迭代器失效！
    }
}

// ✅ 正确：erase-remove 惯用法
v.erase(remove_if(v.begin(), v.end(), predicate), v.end());
八、头文件

cpp
#include <algorithm>  // sort, find_if, count_if, for_each, transform, remove_if

九、算法返回值注意事项

算法	返回值	必须使用
find_if	迭代器	✅ 必须，否则不知道找没找到
count_if	整数	⚠️ 建议使用
sort	void	❌ 不需要
for_each	函数对象	❌ 通常不需要
transform	输出迭代器	⚠️ 通常不需要
remove_if	迭代器	✅ 必须（传给 erase）
十、一句话总结

text
STL 算法是标准库写好的函数，配合 Lambda 使用。
查找用 find_if（要检查返回值）
计数用 count_if
排序用 sort
遍历用 for_each
转换用 transform
删除用 erase + remove_if
